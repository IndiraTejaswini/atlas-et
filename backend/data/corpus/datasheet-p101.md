# Datasheet — P-101A/B Crude Charge Pumps

| Parameter | Value |
|-----------|-------|
| Service | Crude oil charge to E-104 / C-201 |
| Type | Centrifugal, single stage, overhung (Sulzer OHH) |
| Rated flow | 450 m3/hr |
| Differential head | 420 m |
| Discharge pressure | 42 barg |
| Min continuous flow | 130 m3/hr |
| Motor rating | 315 kW, 2980 RPM |
| Seal | Cartridge mechanical seal, API Plan 32 flush |
| Flush fluid | Filtered diesel, 25 micron, +1.5 barg over seal chamber |
| Bearings | SKF 7310 BECBM (thrust) + NU 310 (radial) |
| Lube | ISO VG 68, 4000 hr change interval |
| Casing material | A216 WCB |
| Impeller | CA6NM |
| Design code | API 610 11th Ed |
| Vibration alarm / trip | 7.1 / 11.0 mm/s RMS (ISO 10816) |
